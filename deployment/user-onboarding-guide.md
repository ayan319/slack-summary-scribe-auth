## üîß Troubleshooting

### Common Issues:

#### Upload Problems:
- **File too large**: Reduce file size to under 20MB
- **Unsupported format**: Use PDF or DOCX only
- **Upload fails**: Check internet connection and try again
- **Processing stuck**: Refresh page and check status

#### Slack Connection Issues:
- **Authorization fails**: Check Slack permissions

- **Disconnected**: Reconnect through settings page
- **Missing conversations**: Check channel access permissions

#### Summary Quality:
- **Poor quality**: Rate low and provide feedback
- **Missing content**: Check source document quality
- **Wrong language**: Verify document language settings
- **Incomplete**: Try re-uploading or contact support
